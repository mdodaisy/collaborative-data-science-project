Research Question

How does benthic cover vary across three reef zones of the Belize Barrier Reef?

Visualization of Data

1) Metadata

This dataset contains the metadata for quadrat photos taken every 5m along 50m transects. Six transects were conducted at each of the three study sites in Belize: lagoon (whale shoal), fore reef, and back reef. The dataset contains the image name, site of collection, date of collection, transect number, and quadrat number.

2) Percent Cover Dataset

This dataset consists of percent cover data for each quadrat photo. Percent cover was calculated using CoralNet, a software that automates the analysis of coral reef benthic images. Essentially, CoralNet generates 25 random points over each image, identifies/labels the points, and then calculates percent cover from that data. The dataset contains the image ID, image name (same as above), annotation status (whether the image has been labeled), number of collection points, and various columns corresponding to different benthic categories such as hard coral species, soft coral species, macroalgal species, etc. See the Glossary for a key to each category.

Data Wrangling

Combine the metadata with the percent cover data.

# rename image name column to match that in the metadata
per.cover.data <- per.cover.data %>%
  rename("Image.name" = `Image name`)

# join the two together
benthic.data <- metadata %>%
  left_join(per.cover.data)

Now, we have to calculate the average percent cover for each category at each transect. To do this, we first must gather() our data so that we have a column for the categories and a column for the percent cover of each category.

benthic.data <- benthic.data %>%
  gather(key = category, value = percent_cover, 9:69) # gather all category columns into one column, "category"

Next, we can group by Site, Transect, and Category to calculate the average percent cover for each category at each transect across the ten quadrats.

transect.cover <- benthic.data %>%
  group_by(Site, Transect, category) %>%
  summarize(transect_cover = sum(percent_cover)/1000) # divide by 100 because there are 10 quadrats in a single transect: 100% * 10

This dataset now shows the proportion of each category found in each transect. However, we also want to classify the categories into broad functional groups for downstream analyses.

# classify functional groups
func.group.cover <- transect.cover %>%
  mutate(func_group = case_match(
    category,
    
    # macroalgae
    c("Brown_UFM", "Green_UFM", "Red_UFM", "Dictyota", "Halimeda", "Ulva", "Lobophora", "Sargassum", "Mac_matrix") ~ "Macroalgae",
    
    # corals
    c("AcrCerv", "AcrPalm", "AgAga", "AgTen", "AgUnd", "BL_OrbFav", "BORB", "CoNat", "DiplLab", "FaviaFrag", "MontaCav", "OrbAnn", "OrbFav", "OrbFrank", "PorAstr", "Por_branch", "PorPor", "PseuCliv", "PseuStrig", "Siderastr", "SidSid") ~ "Coral",
    
    # gorgonians
    c("Gorg_Fan", "Gorg_Plume", "Gorg_Whip", "GORG-BRI") ~ "Gorgonian",
    
    # turf
    c("EAM_HR", "EAM_LR", "EAM_RBL") ~ "Turf",
    
    # cyanobacteria
    c("Cyano_HR", "Cyano_LR", "Cyano_RBL", "Cyano_Sand") ~ "Cyanobacteria",
    
    # crustose coralline algae (CCA)
    c("CCA_LR", "CCA_HR", "CCA_RBL") ~ "CCA",
    
    # sponges
    c("Spong_Crus", "Spong_Erec", "Spong_Glo") ~ "Sponge",
    
    # seagrass
    c("SyrFil", "ThalTes") ~ "Seagrass",
    
    # encrusting algae
    c("Brown_Enc", "Red_Enc") ~ "Encrusting_algae",
    
    # abiotic
    c("Sand", "Bare_Rock", "Debris") ~ "Abiotic",
    
    # all other categories
    .default = "Other"))


# calculate percent cover for each functional group in each transect
func.group.cover <- func.group.cover %>%
  select(-category) %>%
  group_by(Site, Transect, func_group) %>%
  summarise(transect_cover = sum(transect_cover))
